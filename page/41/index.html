An alternative to 'var_dump' in PHP

10 Sep 2012

Whilst working with PHP, I seem to use ‘var_dump’ a ridiculous amount, its a debugging must-have. However, it does fall short in a few ways, especially in how it does not take into consideration that the function is almost always displayed in a HTML page - switching to the page’s source can become a pain. Due to the pitfalls a host of projects such as Krumo and Kint have cropped up to cater for these needs. As well as these projects, if you have XDebug installed it will replace the default ‘var_dump’ function with its own implementation that outputs the information with well-needed styling.
