Mounting an OSX SSH Volume using FUSE and SSHFS

Feb 17, 2014

FUSE for OSX is a Mac OSX port of the popular Filesystem in Userspace (FUSE) software, and a successor to MacFUSE. Once installed, you have the ability to mount many different types of remote and local filesystems in user space, without the need to alter the kernel. An example filesystem that I have found to be very useful is SSHFS, which allows you to interact with files located on a remote server via the SSH protocol. Once mounted, the specified remote directory can be accessed as if it were a local volume.
