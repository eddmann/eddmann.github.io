Bringing Back the Blink Tag using JavaScript

29 Feb 2016

Who doesn’t miss the blink tag that was ever-present in years past? I thought it would be fun to quickly code up a JavaScript implementation that replicates the functionality that has ‘sadly’ been removed in modern browsers.

document.addEventListener('DOMContentLoaded', function () {
  setInterval(function () {
    [].forEach.call(document.getElementsByTagName('blink'), function (el) {
      el.style.visibility =
        el.style.visibility === 'visible' ? 'hidden' : 'visible';
    });
  }, 500);
});
