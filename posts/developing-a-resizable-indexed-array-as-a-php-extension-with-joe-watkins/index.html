Developing a Resizable-Indexed Array as a PHP Extension with Joe Watkins

Aug 28, 2015

Following on from our first screencast, which touched upon how to set up a PHP extension development environment and the creation of a simple array_sum-like function. We now further this topic by implementing a resizable-indexed array class which supplies very similar functionality to that of the SplFixedArray class.

Throughout this discussion, we examine the creation of a custom Zend object class. We use this to invoke an implemented data structure located in another file. We implement internal interfaces, object handlers and highlight the importance of PHP extension tests.
